# Google oauth2 test

This is a simple test with express and Google oauth2

### How to start the test

A client id and client secret are already provided for the test. Please replace them with yours to test your own app.

In a terminal, please install dependencies before go:

```
$ yarn install
```

or 

```
$ npm i
```

Then, use nodemon or ts-node to start the app:

```
$ npx nodemon index.ts
```

or 

```
$ npx ts-node index.ts
```
